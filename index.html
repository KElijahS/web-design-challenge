Summary: Latitude vs. X

Overall the point of this project was to see the differences in weather depending on how far it was from the equator. we did this by pull data from OpenWeatherMap API to get the dataset on multiple cities throughout the world in different locations.

After putting together all the data into a dataset we graphed the different aspects of the weather vs. latitude with Matplotlib. The aspects we looked at were temperature, cloudiness, humidity, and windspeed. Going through this website youll see all the data and visualizations used to display our analysis along with our observations based off of that.
